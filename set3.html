Q1) JSP Login Validation

Develop a JSP program to validate user login using username and password.

Project Setup Steps:

  • File → New Project
  • Categories → Java Web
  • Projects → Web Application → Click Next
  • Name and Location → e.g., LoginValidationApp
  • Server → Choose Apache Tomcat
  • Java EE Version → e.g., Java EE 7 Web
  • Finish setup

1. login.jsp

<%@page contentType="text/html" pageEncoding="UTF-8"%>
<!DOCTYPE html>
<html>
<head>
  <title>LOGIN PAGE</title>
</head>
<body>
  <form action="forward.jsp" method="post">
    <h1>Enter your Credentials !</h1>
    <h3>Username: <input type="text" name="uname"/></h3>
    <h3>Password: <input type="password" name="pwd"/></h3>
    <input type="submit" value="SUBMIT">
    <input type="reset"/>
  </form>
</body>
</html>

2. forward.jsp

<%@page contentType="text/html" pageEncoding="UTF-8"%>
<%@page import="java.io.File"%>
<% if(request.getParameter("uname").equals("CSE") && request.getParameter("pwd").equals("griet")) { %>
  <jsp:forward page="welcomePage.jsp"/>
<% } else { %>

  <%@include file="login.jsp"%>
<% } %>

Q2) PHP Matrix Operations

Write a PHP program to find the transpose, multiplication, and addition of two matrices.

<!DOCTYPE html>
<html>
<body>
<?php
function pr($c){
  for($i=0;$i<3;$i++){
    for($j=0;$j<3;$j++){
      echo $c[$i][$j]." ";
    }
    echo "<br>";
  }
  echo "<br>";
}

$a = [[1,1,3],[4,5,6],[7,8,9]];
$b = [[6,8,9],[4,5,6],[4,2,3]];

echo "<b>First Matrix : </b><br>"; pr($a);
echo "<b>Second Matrix : </b><br>"; pr($b);

// Transpose of first matrix
for ($i=0; $i<3; $i++)
  for ($j=0; $j<3; $j++)
    $c[$i][$j] = $a[$j][$i];
echo "<b>Transpose Of First Matrix : </b><br>"; pr($c);

// Transpose of second matrix
for ($i=0; $i<3; $i++)
  for ($j=0; $j<3; $j++)
    $c[$i][$j] = $b[$j][$i];
echo "<b>Transpose Of Second Matrix : </b><br>"; pr($c);

// Addition
for ($i=0; $i<3; $i++)
  for ($j=0; $j<3; $j++)
    $c[$i][$j] = $a[$i][$j] + $b[$i][$j];
echo "<b>Addition Of Two Matrix : </b><br>"; pr($c);

// Multiplication
for ($i=0; $i<3; $i++)
  for ($j=0; $j<3; $j++){
    $c[$i][$j] = 0;
    for ($k=0; $k<3; $k++)
      $c[$i][$j] += $a[$i][$k] * $b[$k][$j];
  }
echo "<b>Multiplication Of Two Matrix : </b><br>"; pr($c);
?>
</body>
</html>
