Task 11: PHP States Program

<html>
<body>
<?php
$states = "Mississippi Alabama Texas Massachusetts Kansas";
$b = explode(' ', $states);
$statesList = array();

// Original logic from your PDF
foreach ($b as $c) {
    $n = strlen($c);
    
    // a. Ends with xas
    if($c[$n-1] == 's' && $c[$n-2] == 'a' && $c[$n-3] == 'x') 
        $statesList[0] = $c;
    
    // b. Starts with K/k, ends with s (case-insensitive)
    if(strtolower($c[0]) == 'k' && strtolower($c[$n-1]) == 's') 
        $statesList[1] = $c;
    
    // c. Starts with M, ends with s
    if($c[0] == 'M' && $c[$n-1] == 's') 
        $statesList[2] = $c;
    
    // d. Ends with a
    if($c[$n-1] == 'a') 
        $statesList[3] = $c;
}

echo "<pre>";
print_r($statesList);
echo "</pre>";
?>
</body>
</html>

Execution Steps:

  1. Save as states.php in XAMPP's htdocs folder
  2. Start Apache server
  3. Access via: http://localhost/states.php
  4. Output will be: 
    Array (
    [0] => Texas
    [1] => Kansas
    [2] => Massachusetts
    [3] => Alabama 
)

Key Features (as per PDF):

  • Manual string analysis without regex
  • Direct character position checks
  • Case-insensitive comparison for 'K' search
  • Array indices 0-3 for different patterns
