Task 5: jQuery AJAX Request

Working Demo:

Complete Code:

HTML (index.html):

<!DOCTYPE html>
<html>
<head>
    <title>jQuery AJAX Demo</title>
    <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
    <style>
        button {
            padding: 10px 15px;
            background: #3498db;
            color: white;
            border: none;
            cursor: pointer;
        }
        #result {
            margin-top: 20px;
            padding: 15px;
            background: #f0f0f0;
        }
    </style>
</head>
<body>
    <button id="load-btn">Load Server Date and Time</button>
    <div id="result"></div>
    
    <script>
        $(document).ready(function() {
            $("#load-btn").click(function() {
                $.get("date-time.php", function(data) {
                    $("#result").html("<strong>Server Response:</strong> " + data);
                });
            });
        });
    </script>
</body>
</html>

PHP (date-time.php):

<?php
// Set the timezone to Asia/Kolkata
date_default_timezone_set("Asia/Kolkata");

// Return current date and time
echo date("F d, Y h:i:s A");
?>

How to Execute:

  1. Create a folder named "task5"
  2. Save the HTML code as "index.html" in this folder
  3. Save the PHP code as "date-time.php" in the same folder
  4. Place the folder on a PHP-enabled server (like XAMPP)
  5. Access through the server (e.g., http://localhost/task5/)
  6. Click the button to see the server's current date and time
