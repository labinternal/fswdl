Task 2: Squares & Cubes Table

Working Demo:

Complete Code:

<!DOCTYPE html>
<html>
<head>
    <title>Squares & Cubes</title>
    <style>
        table {
            border-collapse: collapse;
            margin: 20px auto;
        }
        td, th {
            border: 1px solid #999;
            padding: 10px;
            text-align: center;
        }
        th {
            background-color: #f2f2f2;
        }
    </style>
</head>
<body>
    <h2 style="text-align: center">Number Squares & Cubes (0-10)</h2>
    <table>
        <tr>
            <th>Number</th>
            <th>Square</th>
            <th>Cube</th>
        </tr>
        <script>
            for(let n = 0; n <= 10; n++) {
                document.write(`
                    <tr>
                        <td>${n}</td>
                        <td>${n*n}</td>
                        <td>${n*n*n}</td>
                    </tr>
                `);
            }
        </script>
    </table>
</body>
</html>

How to Execute:

  1. Copy the entire code block
  2. Save as squares.html
  3. Open in any web browser
  4. Automatically displays table with values 0-10
